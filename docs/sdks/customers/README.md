## Overview

## list

Retrieve a list of customers for the authenticated workspace.

### Example Usage

```typescript
import { Pimms } from "pimms";

const pimms = new Pimms({
  token: "PIMMS_API_KEY",
});

async function run() {
  const result = await pimms.customers.list({});

  // Handle the result
  console.log(result);
}

run();
```
