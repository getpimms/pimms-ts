# Folders
(*folders*)

## Overview

### Available Operations

* [create](#create) - Create a new folder
* [list](#list) - Retrieve a list of folders
* [update](#update) - Update a folder
* [delete](#delete) - Delete a folder

## create

Create a new folder for the authenticated workspace.

### Example Usage

```typescript
import { Pimms } from "pimms";

const pimms = new Pimms({
  token: "PIMMS_API_KEY",
});

async function run() {
  const result = await pimms.folders.create();

  // Handle the result
  console.log(result);
}

run();
```
